### Installation

`disperse` can be installed and used like any other modern Python package (although it is currently available only on
GitHub and is not yet published to PyPi). For example, if you use [`pip`](https://pip.pypa.io/en/stable/installation/),
you can run the following command to install `disperse` from GitHub.

``` bash
pip install git+https://github.com/UNM-NuCO/disperse.git
```
